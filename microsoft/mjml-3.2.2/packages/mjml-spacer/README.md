## mjml-spacer

Displays a blank space.

```xml
<mjml>
  <mj-body>
    <mj-container>
      <mj-section>
        <mj-column>
          <mj-spacer height="50px" />
        <mj-column>
      </mj-section>
    </mj-container>
  </mj-body>
</mjml>
```
